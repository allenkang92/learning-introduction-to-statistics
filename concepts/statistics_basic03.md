# 확률 및 확률분포함수

## 확률의 정의

### 확률(probability)의 개념

확률이란 '어떤 사건이 일어날 가능성을 0과 1사이의 실수로 표시'한 것입니다. 다시 말해, 동일한 상태에서 동일한 시행을 무한 번 반복한다고 할 때 궁극적으로 전체 시행 중에서 특정 사건이 발생할 비율을 나타냅니다.

예:
- 공정한 주사위를 던질 때 6의 눈이 나올 확률은?
- 윷놀이를 할 때 윷이 나올 확률은 얼마일까?

### 확률적 실험(통계적 실험)

실험의 결과가 구체적으로 어떤 것인가는 알 수 없지만 전체 가능한 모든 결과들을 알고 있고 반복이 가능한 경우를 **확률적(통계적) 실험**이라고 합니다.

실생활 사례:
1. 어느 생산공정에서 제품을 반복 생산하고 있다. 제품은 정상품 또는 불량품이지만 무엇이 될지는 알 수 없다.
2. 피자를 주문하면 피자 배달 시간은 대개 30분 이내지만 정확히는 알 수 없다.

### 표본공간, 사건

- **표본공간(sample space)**: 통계적 실험이나 조사에서 일어날 수 있는 모든 가능한 결과의 모임 (대개 S로 표시)
- **사건(event)**: 일어날 수 있는 모든 가능한 결과 중에서 특정한 성질을 갖는 결과의 모임 (대개 A, B, C 등으로 표시)

확률실험의 예:
1. 동전을 한 번 던지는 실험
   - 원소: H(= 앞면) 또는 T(= 뒷면)
   - 표본공간: S = {H, T}

2. 주사위를 한 번 던지는 실험
   - 원소: 1, 2, 3, 4, 5, 6
   - 표본공간: S = {1, 2, 3, 4, 5, 6}
   - 사상: 실험의 결과 짝수 눈이 나오는 경우 A = {2, 4, 6}

3. 생산공정 사례
   - 표본공간: S = {정상품, 불량품}
   - 한 개 제품을 검사할 때 불량품일 사건: A = {불량품}

4. 피자 배달 사례
   - 표본공간: S = {t ∶ t > 0}
   - 배달시간이 20분 이내일 사건: A = {t ∶ 0 < t < 20}

### 확률의 고전적 정의

표본공간 내 모든 원소의 발생 가능성이 같은 경우:

- 이산형 표본공간의 경우: P(A) = (사건 A에 포함된 원소의 수) / (표본공간 S의 원소의 수)
- 연속형 표본공간의 경우: P(A) = (사건 A의 크기) / (표본공간 S의 전체 크기)

예제:
1. 주사위를 던지는 실험
   - 표본공간: S = {1, 2, 3, 4, 5, 6}
   - 홀수가 나올 확률: P(A) = 3/6 = 1/2 (A = {1, 3, 5})

2. 피자 배달시간의 예
   - 표본공간: S = [10, 30] ("배달 시간은 10분에서 30분까지 같은 가능성")
   - 20분에서 25분 사이에 배달되는 사건: B = [20, 25]
   - 배달시간이 20분에서 25분 사이일 확률: P(B) = 5/20 = 1/4

### 확률의 상대도수적 정의

사건 A가 발생할 확률(P(A)로 표시)은 같은 조건하에서 통계적 실험을 수없이 많이 반복시행 하였을 때 사건 A가 발생하는 비율(즉, 상대도수)입니다.

P(A) = lim(n→∞) (An/n)

여기서 An은 n번의 시행 중 사건 A가 발생한 횟수입니다.

### 확률의 공리적 정의

콜모고로프(Kolmogorov, 1903-1987)는 확률을 다음과 같이 공리적으로 정의했습니다:

1. 표본공간 S에서 임의의 사건 A에 대하여 0 ≤ P(A) ≤ 1
2. P(S) = 1
3. 서로 배반인 사건 A₁, A₂, ...에 대하여
   P(A₁ ∪ A₂ ∪ ...) = P(A₁) + P(A₂) + ...

## 확률의 계산

### 순열과 조합

이산형 표본공간에서 확률 계산을 위해서 원소의 수를 세는 방법:

**순열(permutation)**: n개의 사물 중 r개를 선택하여 순서를 고려하여 나열하는 방법의 수
- ₙPᵣ = n! / (n-r)!

**조합(combination)**: n개의 사물 중 r개를 순서를 고려하지 않고 추출하는 방법의 수
- ₙCᵣ = n! / (r! × (n-r)!)

### 확률의 덧셈법칙

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

A ∩ B = ∅인 경우 ⇔ "사건 A, B는 서로 배반사건"일 때:
P(A ∪ B) = P(A) + P(B)

예제:
통계·데이터과학과 2학년 학생 40명 중
- 경제학 수강 학생: 25명
- 경영학 수강 학생: 30명
- 두 과목 동시수강 학생: 20명

A: 경제학을 수강하는 사건
B: 경영학을 수강하는 사건
P(A) = 25/40, P(B) = 30/40, P(A ∩ B) = 20/40

경제학 또는 경영학을 수강할 확률:
P(A ∪ B) = P(A) + P(B) - P(A ∩ B) = 25/40 + 30/40 - 20/40 = 35/40

### 조건부확률(conditional probability)

사건 B가 일어났다는 조건하에서 사건 A가 일어날 확률은 다음과 같이 정의됩니다:

P(A|B) = P(A ∩ B) / P(B) (단, P(B) > 0)

예제:
어느 학과에 남자가 30명, 여자가 20명이고, 남자 중 10명, 여자 중 8명이 안경을 착용합니다.
한 명을 뽑았더니 여자였을 때, 이 학생이 안경을 쓴 학생일 확률은?

M: 남자, F: 여자
N: 안경미착용, G: 안경 착용

P(G|F) = P(G ∩ F) / P(F) = (8/50) / (20/50) = 8/20 = 2/5

### 확률의 곱셈법칙

P(A) > 0, P(B) > 0이면,
P(A ∩ B) = P(A)P(B|A) = P(B)P(A|B)

### 독립사건(independent event)

사건 A와 B가 서로 독립사건이라는 것은 다음 세 조건이 동치임을 의미합니다:
- P(A ∩ B) = P(A)P(B)
- P(B|A) = P(B)
- P(A|B) = P(A)

예제:
주사위를 두 번 던지는 실험
- A: 첫 번째 눈이 2가 되는 사건
- B: 두 번 던져 나온 눈의 수의 합이 5인 사건

표본공간: S = {(1,1), (1,2), ..., (6,6)}
A = {(2,1), (2,2), (2,3), (2,4), (2,5), (2,6)}
B = {(1,4), (2,3), (3,2), (4,1)}
A ∩ B = {(2,3)}

P(A) = 6/36 = 1/6, P(B) = 4/36 = 1/9, P(A ∩ B) = 1/36

P(A ∩ B) ≠ P(A)P(B) ⟹ "사건 A와 B는 서로 독립이 아니다"

### 여사건의 확률 계산

A^c를 사건 A의 여사건이라고 할 때:
P(A^c) = 1 - P(A)

예제:
상자 속의 6개 제품 중에서 2개가 불량품이다. 제품검사를 위해 3개를 추출할 때, 적어도 1개의 불량품이 뽑힐 확률은?

A: 적어도 1개의 불량품이 발견될 사건
A^c: 불량품이 하나도 없을 사건

P(A^c) = (4C₃)/(6C₃) = 4/20 = 1/5

P(A) = 1 - P(A^c) = 1 - 1/5 = 4/5

## 확률변수

### 확률변수(random variable)의 정의

확률변수는 표본공간의 각 원소에 실수값을 대응시켜 주는 함수입니다.

- **이산형 확률변수**: 불량품의 수, 고속도로에서의 사고건수, 방문자수 등
- **연속형 확률변수**: 전구의 수명, 몸무게, 체온, 통근시간 등

예제:
1. 동전을 두 번 던지는 실험
   X = "두 번 던질 때 나온 앞면의 수"
   - {앞, 앞} → 2
   - {앞, 뒤} → 1
   - {뒤, 앞} → 1
   - {뒤, 뒤} → 0
   
   P(X = 2) = P({앞앞}) = 1/4
   P(X = 1) = P({앞뒤, 뒤앞}) = 2/4
   P(X = 0) = P({뒤뒤}) = 1/4

2. 200가구를 조사대상으로 지난 일년 동안 각 가구에서 병원 방문 회수 조사

### 이산형 확률분포의 성질

확률분포함수 p(x) = P(X = x)에 대하여:
1. 0 ≤ p(x) ≤ 1
2. Σp(x) = 1
3. P(a < X ≤ b) = Σ₍ₓ: a<x≤b₎p(x)

### 연속형 확률변수와 확률밀도함수

연속형 확률변수 X에 대한 P(a ≤ X < b)는 해당 구간에서 확률밀도함수 f(x)의 적분으로 구합니다:

P(a ≤ X ≤ b) = ∫[a,b] f(x)dx

**확률밀도함수 f(x)의 성질**:
1. f(x) ≥ 0
2. ∫[-∞,∞] f(x)dx = 1
3. P(a < X ≤ b) = ∫[a,b] f(x)dx

## 기댓값과 분산

### 확률변수 X의 기댓값(평균)

확률변수 X의 확률밀도함수를 f(x)라고 하면:

- 이산형:
  - E(X) = μ = Σ xᵢf(xᵢ)
  - E(g(X)) = Σ g(xᵢ)f(xᵢ)

- 연속형:
  - E(X) = μ = ∫[-∞,∞] xf(x)dx
  - E(g(X)) = ∫[-∞,∞] g(x)f(x)dx

예제:
어느 자동차 판매영업소의 1주일간 자동차 판매대수 확률분포
'X = 1주일간 판매대수'

E(X) = Σ xᵢf(xᵢ)
     = 0 × 0.1 + 1 × 0.1 + 2 × 0.2 + 3 × 0.3 + 4 × 0.2 + 5 × 0.1
     = 2.7

### 확률변수 X의 분산

Var(X) = σ² = E[(X - μ)²]

확률변수 X의 확률밀도함수를 f(x)라고 하면:

- 이산형: Var(X) = Σ(xᵢ - μ)²f(xᵢ) = E(X²) - [E(X)]²
- 연속형: Var(X) = ∫[-∞,∞] (x - μ)²f(x)dx = E(X²) - [E(X)]²

σ = √Var(X): 표준편차

### 새로운 확률변수 aX + b

새로운 확률변수 aX + b의 기댓값과 분산(a와 b는 임의의 상수):

- E(aX + b) = aE(X) + b
- Var(aX + b) = a²Var(X)

### 표준화된 확률변수(standardized random variable)

평균이 μ, 표준편차가 σ인 확률변수 X에 대해서:

- 표준화된 확률변수: Z = (X - μ) / σ
- 확률변수 Z의 평균은 0, 분산은 1

## 요약

- 확률은 어떤 사건이 일어날 가능성을 0과 1 사이의 실수로 표현한 것.
- 확률적 실험은 결과를 정확히 예측할 수 없지만 가능한 모든 결과를 알고 있고 반복 가능한 실험.
- 확률계산에는 순열과 조합, 확률의 덧셈법칙, 조건부확률, 확률의 곱셈법칙, 독립사건, 여사건의 확률 등 여러 방법이 있음.
- 확률변수는 표본공간의 각 원소에 실수값을 대응시키는 함수로, 이산형과 연속형으로 나뉨.
- 기댓값은 확률변수의 평균적인 값을, 분산은 확률변수가 기댓값으로부터 퍼져있는 정도를 측정.
- 표준화된 확률변수는 원래 확률변수를 평균이 0, 분산이 1인 형태로 변환한 것.
